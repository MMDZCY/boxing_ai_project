# 🥊 Boxing AI Trainer - 拳击动作智能训练系统

基于 YOLOv8 姿态识别的拳击动作实时评估、量化打分与偏差分析系统。

## 功能特性
- ✅ **人体姿态提取**：基于 YOLOv8n-pose 的实时关键点检测
- ✅ **动作量化评估**：基于拳击动力链的专项打分模型（肘/髋/重心）
- ✅ **DTW 时序对齐**：解决动作时长不一致的逐帧对比问题
- ✅ **实时摄像头反馈**：自动检测出拳，即时打分
- ✅ **历史记录追踪**：训练得分趋势可视化
